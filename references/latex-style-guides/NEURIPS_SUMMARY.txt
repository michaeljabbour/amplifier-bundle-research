KEY FORMATTING RULES:
- Maximum 9 pages of content (excluding references, appendices)
- Font: Times New Roman, 10pt
- Line spacing: 11pts
- Text area: 5.5" wide x 9" tall
- Citations: Author/Year OR Numeric (must be consistent)
- Default bibliography: natbib package

PACKAGE OPTIONS:
- Default (submission): \usepackage{neurips_2024}
- Final/Camera-ready: \usepackage[final]{neurips_2024}
- ArXiv preprint: \usepackage[preprint]{neurips_2024}
- If natbib conflicts: \usepackage[nonatbib]{neurips_2024}

IMPORTANT:
- Always use the current year's style file
- Do NOT modify the .sty file (grounds for rejection)
- Omit 'final' and 'preprint' options at submission time
- Maintain anonymity by using "Anonymous submission" for authors
- Include acknowledgments only in final camera-ready version
- All co-authors must have OpenReview profiles
